# HedgeDoc
## Аннотация
**HedgeDoc** – это open-source платформа для совместного редактирования документов в режиме реального времени, фокусирующаяся на поддержке Markdown. С ее помощью пользователи могут одновременно создавать и редактировать документы, видеть правки друг друга в реальном времени и обмениваться заметками или протоколами встреч. 

Основные особенности HedgeDoc:
- Поддержка Markdown с подсветкой синтаксиса.
- Возможность вставки изображений, графиков и даже математических формул с использованием LaTeX.
- Поддержка эмодзи и расширенной разметки.
- Реальное время и общий доступ к документам через ссылки, что делает его удобным для совместной работы в командах.

Проект идеально подходит для ведения заметок, документации и работы над текстами совместно с другими пользователями.

## Инструкция по запуску

`make up`

Открыть в браузере [http://localhost:80](http://localhost:80)
