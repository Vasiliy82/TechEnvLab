# TechEnvLab

Добро пожаловать в **TechEnvLab** (расшифровывается как Technology Environment Laboratory, Лаборатория Технологических Сред) – репозиторий для демонстрации технологий и паттернов развертывания через Docker и Compose! Цель этого проекта – создать набор готовых окружений для быстрой демонстрации возможностей различных технологий и инструментов. **TechEnvLab** станет вашей витриной для изучения, настройки и тестирования известных стэков.

## О проекте

В **TechEnvLab** вы найдете:
- **Docker Compose файлы**, **конфигурации** и **скрипты** для настройки и развертывания разнообразных демо-сред;
- Поддержку популярных технологий и инструментов, таких как PostgreSQL, Redis, Kafka, а также специализированных решений (например, кластерных настроек);
- Структуру репозитория, позволяющую легко находить нужные конфигурации по категориям, версиям и паттернам развертывания.

Проект ориентирован на:
- **Архитекторов** и **разработчиков**, которые хотят быстро развернуть демо-окружение для тестирования идеи или исследования технологии;
- **Инженеров по DevOps** и **системных администраторов**, которые ищут примеры для настройки инструментов и интеграций;
- Всех, кто интересуется технологиями и хочет разобраться в их развертывании и настройке.

## Как это работает

Мы структурировали репозиторий так, чтобы было удобно находить и запускать нужные демо. Вы можете использовать следующую структуру для навигации:

- **Инструменты**: все окружения разделены на категории по основным направлениям – базы данных, кэши, очереди, балансировщики.
- **Версии и паттерны**: выберите нужную версию инструмента и паттерн развертывания (например, кворумная репликация, кластеризация, режим Standalone).

Пример:
```
db/postgresql-17/standalone/compose.yml   # Standalone PostgreSQL 17
db/postgresql-15/citus/compose.yml        # PostgreSQL 15 с Citus
db/postgresql-17/quorum-sync/compose.yml  # Кластер PostgreSQL 17 с кворумной репликацией
```

## Быстрый старт

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Vasiliy82/TechEnvLab.git
   ```
2. Перейдите в нужную папку и запустите окружение:
   ```bash
   cd db/postgresql-17/standalone
   docker-compose up -d
   ```
3. **Исследуйте!** – смотрите логи, экспериментируйте с конфигурациями и учитесь.
